# Luy Tracker (React)

This project has been migrated from plain HTML/CSS/JavaScript to a React + Vite structure.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Cache compatibility

The app still uses the same `localStorage` keys as before:

- `expenses`
- `categories`
- `language`
- `theme`

Your existing cached records remain compatible after migration.
